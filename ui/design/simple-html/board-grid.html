לוח 5×5 · Board grid skeleton

Bare 5×5 layout. Grid + gap (never inline flow) so drag-reorder / delete survive edits. On mobile the same grid renders inside a pan/zoom viewport at ~720px width (see DESIGN.md · Mobile board). Colors are illustrative — real roles come from the key card.

Cell recipe: .tile = cream face + role ring via --r. Punch hole top-center, role icon+shape top-left, white label strip at the bottom. For the revealed/chosen and selected states see card-states.html. Numbered clue badges go top-right (RTL-leading corner).
